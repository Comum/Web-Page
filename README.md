# Web-Page
My personal web page

## How to test

1. Run ``` npm install live-server -g ```
2. cd to the folder
3. Run ``` live-server ```
4. Go to ``` http://localhost:8080 ```

### Technologies used

1. HTML5 and CSS3
 - No precompilers, only CSS
2. Vanilla and jQuery
 - No frameworks, all done from scratch
3. Next step: integrate WebPack
